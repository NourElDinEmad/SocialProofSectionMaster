### What I learned
Flex box is awesome "CSS" property
```css
  .rates>div{
    position: relative;
    display: flex;
    gap: 10px;
  }
  .rates>div:nth-of-type(1){
    align-self: flex-start;
  }
  .rates>div:nth-of-type(2){
    align-self: center;
  }
  .rates>div:nth-of-type(3){
    align-self: flex-end;
  }
```
